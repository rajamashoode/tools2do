What is the Passport Size Photo Resizer?

The Passport Size Photo Resizer helps you quickly create an image with the right dimensions for passport-style photos and online forms. Instead of opening heavy editing software, you can upload a picture, enter the width and height, resize it in your browser and download the result.

How to use this image resizer

  2. Enter the required width and height in pixels.
  3. Click Resize Image.
  4. Download the resized image and use it where needed.
