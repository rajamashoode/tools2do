Meta description length guide

Meta descriptions appear in search results when Google chooses to show them. They should summarize the page clearly and encourage clicks without keyword stuffing.

There is no fixed guaranteed display length, but concise descriptions around 140–160 characters are often easier to scan. Always make the description useful for humans first.
