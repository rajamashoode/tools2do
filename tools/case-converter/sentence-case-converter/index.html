What is Sentence Case?

Sentence Case is a capitalisation style where only the first letter of the first word in each sentence is capitalised, along with proper nouns. It is the standard style for body text, email writing, social media posts and informal content. This tool converts text in any case — ALL CAPS, Title Case, or mixed — to clean sentence case instantly.

How to convert to Sentence Case

  1. Paste your text into the input box above.
  2. Click Convert to Sentence Case.
  3. The output will be lowercase with the first word of each sentence capitalised.
  4. Copy the result for use in your document or post.

Frequently asked questions

Does this tool preserve proper nouns?

The tool capitalises the first word of each sentence based on punctuation. It does not automatically detect proper nouns — you may need to manually re-capitalise names after conversion.

What is the difference between sentence case and lowercase?

Lowercase converts everything to small letters. Sentence case converts to lowercase but capitalises the first word after each sentence-ending punctuation mark (. ! ?).

Can I use this for fixing all-caps text?

Yes. Paste your ALL CAPS text, click convert, and the tool will output proper sentence case.
