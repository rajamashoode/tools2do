How to use this tool

  1. Click the file input and select one or more PDF files.
  2. To merge: select multiple PDFs and click Merge PDFs.
  3. To split: enter the page numbers (e.g. 1,3,5-7) and click Split First PDF.
  4. Click Download PDF to save the result.
